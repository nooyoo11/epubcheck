โฟลเดอร์นี้มีแหล่งที่มาของโครงการ epubcheck EpubCheck เป็นเครื่องมือในการตรวจสอบไฟล์ IDPF Epub มันสามารถตรวจจับหลายประเภทของความผิดพลาดใน Epub โครงสร้างภาชนะ OCF, OPF และ OPS เครื่องหมายขึ้นและความสอดคล้องอ้างอิงภายในมีการตรวจสอบ EpubCheck สามารถใช้เป็นเครื่องมือบรรทัดคำสั่งแบบสแตนด์อโลนติดตั้งโปรแกรมประยุกต์บนเว็บหรือใช้เป็นห้องสมุด โครงการบ้าน Epubcheck: http://code.google.com/p/epubcheck/ อาคารเพื่อสร้าง epubcheck จากแหล่งที่คุณต้องการ Java Development Kit (JDK) 1.5 หรือสูงกว่าและ Apache Maven (http://maven.apache.org/ ) 2.3 หรือติดตั้งดังกล่าวข้างต้น Run: mvn ติดตั้ง
